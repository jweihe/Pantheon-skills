# Experiment Case: Skill Forge Basic

We repeatedly build small frontend tools and forget the same checks:

- inspect existing package scripts before adding dependencies
- avoid landing pages when the request is for an app
- start a dev server after implementation
- verify desktop and mobile layout
- summarize files changed and tests run

Create a Codex skill that helps future agents build practical frontend tools with these habits. It should include a validation checklist and avoid becoming a generic frontend tutorial.
